L'importance d'un bon mot de passe



Les mots de passe constituent la première ligne de défense contre l'accès non autorisé à votre ordinateur et vos informations personnelles.
Le plus fort est votre mot de passe, le plus votre ordinateur sera protégé contre les pirates informatiques et les logiciels malveillants.
Tester la complexité de votre mot de passe :

Vous allez tester à quel point votre mot de passe est efficace, plus celui-ci sera complexe (caractères spéciaux, majuscules, chiffres...), plus celui-ci sera dur à cracker.

Exercice "faux profil" :




Avec cet exercice, vous allez voir via un faux profil qu'il est facile d'usurper l'identité de quelqu'un.
Mise en contexte. Une personne malveillante à créé un faux profil pour se faire passer pour quelqu'un d'autre, la victime.
Elle a trouvé des informations sur la victime, à partir de son compte instagram
Allez vous réussir à rerouver ce faux profil ?
Les informations à votre disposition :
Il se fait passer pour un étudiant à Roanne.
Il s'agit d'un des 3 comptes proposés :

  • paul_trc42

  • _lil_diab_

  • baptiste_brdnv

    • Pour trouver le faux compte, cherchez si les photos postées, likes, abonnés, commentaires ou encore la photo de profil est assez crédible.
    Choisissez le nom du compte que vous pensez être le faux-profil que nous avons créer ci-dessous pour voir si il s'agit du bon profil :

    _lil_diab_
    paul_trc42
    baptiste_brdnv


    Si vous n'avez pas trouvé du premier coup, cela peut-être normal, l'objectif d'une personne usurpant une identité est de paraître pour quelqu'un de normal ou même plus "vraie" que la personne qu'elle usurpe.

    Si vous avez trouvé, c'est peut-être que vous avez le bon oeil ?
